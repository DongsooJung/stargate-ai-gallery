## 제외 방법 (공개 전)

```powershell
# 특정 파일 삭제 (예: 조직도)
Remove-Item "files\org-*.html", "files\thumbs\org-*.png"

# data.json에서도 해당 항목 수동 제거
# 또는 아래 Python 한 줄로 일괄 제거
python -c "import json; d=[x for x in json.load(open('data.json',encoding='utf-8')) if not x['id'].startswith('org-')]; json.dump(d,open('data.json','w',encoding='utf-8'),ensure_ascii=False,indent=1)"
```

## 대안: 비공개 저장소 + Pages

Private 저장소도 GitHub Pages 배포 가능 (Pro 플랜 이상). 이 경우 URL을 알아도 GitHub 로그인 + 저장소 권한 있는 사용자만 접근 가능합니다.

2. **내부 아카이브**: 법인사업·브리핑·Claude 분석 리포트 포함 전체는 Private 저장소 `stargate-internal-gallery`
